Absence Management In case of a prolonged absence (e.g. maternity leave, sick leave, or other extended leave) please report it here. Return to this page to end a current absence, or cancel an upcoming absence. If you are leaving a position and have no further need to use PharmaNet, only select the start date and leave the end date empty.

If you take a position in future that requires you to use PharmaNet, return to PRIME and indicate the end of your absence, and your GPID can be used again immediately.
